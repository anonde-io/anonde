# Code of Conduct

The anonde project adopts the [Contributor Covenant, version 2.1][cc21]
as its code of conduct. By participating in this project — as a
contributor, maintainer, issue reporter, or community member — you
agree to abide by its terms.

The full text, including the pledge, standards, enforcement
responsibilities, scope, and enforcement guidelines, is available at:

- <https://www.contributor-covenant.org/version/2/1/code_of_conduct/>
- Translations: <https://www.contributor-covenant.org/translations>

## Scope

This Code of Conduct applies within all project spaces — the GitHub
repository (issues, pull requests, discussions, code review), any
official chat or mailing list, and any public space when an individual
is representing the project or its community.

## Reporting

If you experience or witness behavior that violates this Code of
Conduct, please report it privately to the maintainers at
<conduct@anonde.io>.

All reports will be reviewed and investigated promptly and fairly.
Maintainers are obligated to respect the privacy and security of the
reporter of any incident.

Security vulnerabilities are handled separately — see
[`SECURITY.md`](SECURITY.md).

## Enforcement

Maintainers will follow the [Community Impact Guidelines][cc21-enforce]
from Contributor Covenant 2.1 when determining the consequences of any
action they deem in violation of this Code of Conduct.

[cc21]: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
[cc21-enforce]: https://www.contributor-covenant.org/version/2/1/code_of_conduct/#enforcement-guidelines
